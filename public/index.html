Why you'll love xZero

  • Highly multi-threading scalable
  • Fast dynamic configuration language
  • Advanced load balancing
  • HTTP/1.1 including pipelining compliance
  • Thin and clean Core API
  • Instant-mode, configurationless deamon
  • Dynamic content compression
  • SSL Support
  • SSL with SNI
  • Basic auth
  • User directory support
  • Directory listing support
  • Custom error pages
  • zero-copy networking optimization
  • FastCGI / CGI support
  • apache-style access log
  • browser match support
  • and many more…
